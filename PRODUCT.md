# Product

## Register

product

## Users

ผู้สร้าง portfolio ชาวไทย — ทั้ง developer, designer, freelancer ที่ต้องการแสดงผลงานออนไลน์แบบเร็วและง่าย

## Product Purpose

PortGen คือ SaaS สร้าง portfolio สำเร็จรูป ผู้ใช้เลือก template, ใส่ข้อมูล, publish ได้ทันที ไม่ต้องเขียนโค้ด

## Brand Personality

- **Voice:** ยืนยัน, ซื่อสัตย์, เป็นกันเอง
- **Tone:** ต้อนรับ, ไว้วางใจได้, ไม่ยัดเยียด
- **3 words:** ง่าย, ดูดี, วางใจได้

## Anti-references

- ไม่ใช่ AI-slop design (gradient text, glassmorphism, สีม่วงบนมืด)
- ไม่ใช่ SaaS ทั่วไปที่ดูเหมือนกันหมด
- ไม่ใช่ Canva-like editor — นี่คือ builder ที่เร็วและจริงจัง
- ไม่ใช่โหลดข้อมูลเยอะเกิน — portfolio ต้องโหลดเร็ว

## Design Principles

1. **Portfolio สวย = คนสร้างดูดี** — design ต้องทำให้ผู้ใช้ดูดีในสายตาคนอื่น
2. **ทำให้ง่ายที่สุด** — ทุก step ต้องกดได้เลย ไม่ต้องคิด
3. **โชว์ผลงาน ไม่ใช่โชว์เครื่องมือ** — UI ของ builder ต้องเงียบ ไม่แย่งฟอร์ม
4. **Thai-first** — ทุกอย่างเป็นภาษาไทย ไม่มี i18n switcher

## Accessibility & Inclusion

- WCAG AA minimum
- Dark mode support throughout
- Touch targets ≥ 44px
